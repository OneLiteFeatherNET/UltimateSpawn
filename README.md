**Installation (plug and play):**  
You will receive a .jar file. You have to put the file into your plugins folder. Run your server -> **done!**<br/>

**Functions:**
- Set multiple spawns
- Remove spawns
- Player join at spawnpoint
- Respawn at spawnpoint
